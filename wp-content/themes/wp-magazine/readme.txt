== Description ==
WP Magazine WordPress Theme is easy to use theme for Passionate Bloggers, Writers and Editors. This theme is developed with lots of options that a magazine newspaper or blog need. WP Magazine Theme can be used by lifestyle bloggers, Personal blogging, magazine, newspaper, news portal and many more. This themes has  multiple layout options for header, footer, featured news, sliders, category news. You will also get unlimited google fonts, color options, easy to use customizer and most importantly show/hide and drag/drop options for all the sections.


== Frequently Asked Questions ==

	= How to install the theme =
		* In your admin panel, go to Appearance > Themes and click the Add New button.
		* Click Upload and Choose File, then select the wp-magazine.zip file. Click Install Now.
		* Click Activate to use your new theme right away.
		* Go to Customiser -> Home Page Setting -> Select - Static Front Page
		* Create categories, pages and post
		* Select category needed for the homepage

2. Click Upload and Choose File, then select the wp-magazine.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Go to Customiser -> Home Page Setting -> Select - Static Front Page
